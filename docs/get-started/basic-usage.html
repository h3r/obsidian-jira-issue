Basic usage

The plugin provides several components to display Jira issues in your notes. You can find details about each component in the dedicated section Components.

The simplest way to insert a reference to a Jira issue is to use the inline syntax. Open a note and use the prefix JIRA: followed by the issue key to add a reference to the issue.
